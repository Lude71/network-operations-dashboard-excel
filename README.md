# Telecom Network Performance Dashboard

## Overview
This project is a telecom data analysis dashboard developed in Microsoft Excel to monitor and evaluate network performance across different regions.

The dashboard provides insights into:
- Network signal strength
- Active users
- Data consumption
- Downtime trends
- Tower performance analysis

## Dashboard Features
- Interactive KPI Cards
- PivotTable & PivotChart Analysis
- Dynamic Slicers for Filtering
- Tower Search System using VLOOKUP
- Signal Health Classification
- Conditional Formatting Alerts
- Telecom Performance Insights

## Excel Skills Applied
- Data Cleaning & Organization
- Dashboard Design
- PivotTables & PivotCharts
- IF Statements
- COUNTIF / SUMIF
- VLOOKUP
- Conditional Formatting
- Data Visualization

## Project Goal
The aim of this project was to simulate a real-world telecom monitoring dashboard while improving practical Excel and data analytics skills.

## Key Observations
- Some regions recorded significantly higher data traffic
- Weak signal towers were identified using automated classification
- Network downtime patterns varied across locations
- Interactive filtering improved performance monitoring
